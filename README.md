# 🧠 Laboratório de Personalidade

*Autoconhecimento através da análise de dados*

---

# 📖 Sobre o projeto

O Laboratório de Personalidade é um projeto desenvolvido para explorar como Inteligência Artificial, Análise de Dados e o modelo Big Five (OCEAN) podem ser utilizados para promover autoconhecimento e desenvolvimento pessoal.

A proposta é transformar conceitos científicos sobre personalidade em uma experiência intuitiva, permitindo que o usuário conheça melhor suas características comportamentais e reflita sobre seus pontos fortes e oportunidades de desenvolvimento.

O projeto também demonstra a aplicação de IA Generativa no desenvolvimento de soluções voltadas para experiência do usuário, análise de comportamento e documentação técnica.

---

# 💡 Como surgiu a ideia

A ideia nasceu durante meus estudos sobre Inteligência Artificial e Ciência de Dados.

Percebi que muitas pessoas possuem dificuldade para compreender seus próprios comportamentos e, principalmente, explicar seus diferenciais em entrevistas de emprego ou processos de desenvolvimento profissional.

A partir dessa reflexão surgiu a proposta de criar um ambiente onde tecnologia e ciência comportamental trabalham juntas para incentivar o autoconhecimento de forma acessível e visual.

---

# 🎯 Objetivos

O Laboratório de Personalidade busca:

- Promover autoconhecimento.
- Auxiliar na identificação de pontos fortes.
- Incentivar o desenvolvimento de competências.
- Demonstrar aplicações práticas de IA Generativa.
- Explorar conceitos de Ciência de Dados aplicados ao comportamento humano.

---

# 🧬 Modelo utilizado

O projeto utiliza como referência o modelo **Big Five (OCEAN)**, composto pelos cinco grandes fatores de personalidade:

- 🌊 **O** - Openness (Abertura à Experiência)
- 📋 **C** - Conscientiousness (Conscienciosidade)
- 🎉 **E** - Extraversion (Extroversão)
- 🤝 **A** - Agreeableness (Amabilidade)
- 💙 **N** - Neuroticism (Neuroticismo)

# ⚠️ Aviso

Este projeto possui finalidade exclusivamente educacional e de desenvolvimento tecnológico.

Não realiza diagnósticos psicológicos, clínicos ou terapêuticos.

Os resultados apresentados servem como ferramenta de reflexão e autoconhecimento, não substituindo avaliações realizadas por profissionais habilitados.
