# Novas ideias 

A interface do usuário precisa de novo design para ficar mais atraente.

pensei em alterar para cores mais alegres,  botões divertidos. O intuito é não deixar o processo entediante e tornar uma ferramenta altamente atrativa.

## Prompt

Quero reformular a landing page inicial (a tela de apresentação do teste) para deixá-la muito mais alegre, vibrante e convidativa, trazendo a energia visual do ambiente /dev!

Por favor, aplique as seguintes mudanças de UI/UX:

1. Cabeçalho e Título:
- Adicione uma pílula/badge no topo com estilo arredondado e gradiente suave com o texto "✨ JORNADA DE AUTOCONHECIMENTO".
- Torne o título principal mais convidativo: "Descubra sua verdadeira personalidade".
- Encurte o texto explicativo para 2 linhas com uma leitura leve.

2. Cards Informativos (Modelo, Perguntas, Duração):
- Mude o fundo dos 3 cards para um tom branco suave/translúcido com bordas finas e arredondadas.
- Adicione um ícone colorido e divertido no topo de cada card (ex: 🧠 para Modelo, 📝 para Perguntas, ⏱️ para Duração).

3. Campo de Nome e Ação Principal (CTA):
- O campo "Como podemos te chamar?" deve ter um input branco com ícone de usuário interno (👤) para dar bom contraste.
- Adicione um botão principal largo (Full Width) bem vibrante e chamativo em gradiente (Laranja para Rosa/Roxo, parecido com o estilo do botão Modo Caos do /dev) com o texto: "Iniciar Minha Jornada ➔".

4. Cores e Clima Geral:
- Substitua a paleta totalmente bege por uma atmosfera mais alegre, fresca e moderna, mantendo um fundo leve pero iluminado.
